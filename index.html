Getting Your Anthropic API Key

This tool uses Claude AI from Anthropic to analyze grant applications. You'll need an API key to get started. Your API key is stored locally in your browser and is only sent directly to Anthropic's servers.

Create an Anthropic Account: Go to console.anthropic.com and sign up for a free account
Navigate to API Keys: Once logged in, click on "API Keys" in the left sidebar or go directly to console.anthropic.com/settings/keys
Create a New Key: Click "Create Key", give it a name (e.g., "NOFA Evaluator"), and copy the key that appears
Add Credits: New accounts include some free credits. For additional usage, add credits at console.anthropic.com/settings/billing
Enter in the Tool: Open the NOFA AI Evaluator, click "Settings", and paste your API key in the "Anthropic API Key" field
